# Libretto-Universitario
Proietta la media ponderata universitaria, aggiorna dopo ogni esame e archivia gli esami accettati. Funziona offline e tiene i dati solo sul dispositivo (niente server, niente account).  Pensata per il sistema italiano: voti in trentesimi, CFU, idoneità escluse dalla media, voto base di laurea in centodecimi.
